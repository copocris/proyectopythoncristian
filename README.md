# Comprobador de Palíndromos
Cristian Fernandez Cornejo 30/11/2025

# Objetivo
Este proyecto es una pequeña aplicación en Python cuyo objetivo es comprobar si una palabra o frase es un palíndromo, incluso si contiene:
    -Mayúsculas y minusculas
    -Espacios
    -Tildes o caracteres acentuados

# Ejecutar el programa
python src/main.py

# Ejecutar tests
python -m unittest tests.test_modulo1 -v
